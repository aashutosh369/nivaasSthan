**Nivashasthan – Property Rental Platform**

**Project Description**

    ↣ Nivashasthan is a full-stack property rental web application built using the MERN stack.
    
    ↣ It allows users to securely register, log in, and manage property listings with image uploads.
    
    ↣ The platform focuses on secure authentication, clean REST API architecture, responsive UI design, and cloud-based image storage.

**Tech Stack**

    ⁕ Frontend
    
        React.js
        
        React Router
        
        Tailwind CSS
        
        Axios
   
   ⁕ Backend
        Node.js
        Express.js
        RESTful APIs    
   ⁕ Database
        MongoDB
        Mongoose
        MongoDB Atlas
   ⁕ Authentication & Security
        JWT (JSON Web Tokens)
        HTTP-only Cookies
        Protected Routes
   ⁕ Cloud Services
        Cloudinary (Image Upload & Storage)

**Key Features**
     ↣ Secure user authentication using JWT and HTTP-only cookies
     ↣ Create, update, delete, and view property listings (CRUD operations)
     ↣ Cloudinary integration for efficient image upload and management
     ↣ Fully responsive UI built with Tailwind CSS
     ↣ Dynamic frontend-backend data integration
     ↣ Protected routes for authorized users

**How It Works**
     ↣ Users register and log in securely.
     ↣ After authentication, users can:
     ↣ Add new property listings
     ↣ Upload property images
     ↣ Edit existing listings
     ↣ Delete their listings
     ↣ Property data is stored in MongoDB.
     ↣ Images are stored securely in Cloudinary.
     ↣ Authentication is handled via JWT stored in HTTP-only cookies.

**Installation Guide**
  Clone the repository
    git clone (https://github.com/aashutosh369/nivaasSthan)
    cd nivashasthan
  Backend Setup
    cd backend
    npm install
    npm run dev
  Create a .env file inside backend folder:
    MONGO_URI=your_mongodb_connection
    JWT_SECRET=your_secret_key
    CLOUDINARY_NAME=your_cloudinary_name
    CLOUDINARY_API_KEY=your_api_key
    CLOUDINARY_API_SECRET=your_api_secret

  Frontend Setup
    cd frontend
    npm install
    npm start
